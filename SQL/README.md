# Blinkit Business Analytics Project

## Overview
This project analyzes Blinkit's sales performance, delivery efficiency, inventory management, customer satisfaction, demand patterns, and promotional strategies using SQL.

## Tools Used
- PostgreSQL
- Excel

## Dataset
- 13,000 records
- 12,617 unique products

## Business Questions
1. Top revenue-generating products
2. Highest-revenue category analysis
3. Delivery performance by city
4. Profitability and customer satisfaction analysis
5. Customer ratings vs discounts
6. Delivery performance vs customer satisfaction
7. Shelf-life analysis
8. Inventory reordering analysis
9. Demand and stock analysis
10. Promotional offer analysis

## Key Skills Demonstrated
- SQL Queries
- Aggregation Functions
- CASE Statements
- Subqueries
- Business Analysis
- Data Interpretation
